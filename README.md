# Namma-Skill 🎓

An Android application developed using Kotlin and Google Gemini AI that connects rural youth with government skill development opportunities.

## About the Project
Namma-Skill is a Training Opportunity Tracker that helps rural youth discover and enroll in government vocational training programs like Electrician, Sewing, Coding, Welding and Mobile Repair.

## Features
- 📚 Course Finder with trade and duration filters
- 🗺️ Center Map using Google Maps
- 📝 One-click Apply with AI-generated Candidate Summary
- 🌟 Success Stories of course completers
- 🔔 Push Notifications for new batch alerts
- 📍 Interest Ping for trainer callback
- 🔥 Firebase Realtime Database integration

## Technologies Used
- Kotlin

## Setup Instructions
1. Clone the repository
2. Open in Android Studio
3. Add your google-services.json file
4. Add your Gemini AI API key in ApplyActivity.kt
5. Build and run the app
